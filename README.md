# Credit Risk Dataset Analysis on SQL

This project performs a comprehensive credit risk analysis using SQL to query and analyze loan portfolio performance, vintage analysis, credit rating migrations, and macroeconomic stress scenarios. The analysis combines loan-level data with vintage performance metrics to assess credit risk across different dimensions including sector, origination vintage, and macroeconomic conditions.

The raw datasets pulled from kaggle can be found here:

| File Name | Description | Link |
|-----------|-------------|------|
| credit_ratings.csv | Provides a comprehensive view of a credit portfolio, combining loan-level details, credit rating migrations, macroeconomic stress scenarios, portfolio performance metrics, and vintage analysis. | [View File](./data/raw/credit_ratings.csv) |
| loan_portfolio.csv |  Contains detailed information on individual loans in the portfolio. | [View File](./data/raw/loan_portfolio.csv) |
| macro_stress_scenarios.csv | Simulates the impact of different macroeconomic shocks on portfolio risk metrics. | [View File](./data/raw/macro_stress_scenarios.csv) |
| portfolio_metrics.csv | A time-series portfolio that shows the monthly aggregated portfolio metrics from 2015 to 2024. | [View File](./data/raw/portfolio_metrics.csv) |
| vintage_analysis.csv | Tracks the performance of loans originated in each quarter over time.| [View File](./data/raw/vintage_analysis.csv) |
